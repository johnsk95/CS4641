How to run my code

1. Download or clone Project4 from the address below
https://github.com/johnsk95/CS4641/tree/master/Project4

2. cd to RL_sim

3. Execute command
java -jar rl_sim.jar

4. Click 'Value Iteration' 'Policy Iteration' 'Q Learning' to run the algorithms in the gridworlds

5. In each simulation windows, click 'LoadMaze' button to load the mazes to simulate
    The small maze is small.maze
    The big maze is 7_medium_complex_multigoal.maze

6. Fill in the appropriate parameters and press 'Initialize'. Then press 'Execute' to see the final result

* Stats for the simulations are in the stats.xlsx file.
